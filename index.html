Uma introdução ao HTML e CSS.

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ele é usado para estruturar o conteúdo e definir a semântica dos elementos em uma página.

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para descrever a apresentação de documentos HTML. Ele controla o layout, as cores, as fontes e outros aspectos visuais de uma página.
